Privacy Policy – WeTalk

App Name: WeTalk
Developer: Milan Fataniya

1. Introduction

WeTalk respects your privacy. This Privacy Policy explains how information is collected, used, stored, and protected when you use the WeTalk mobile application.

By using WeTalk, you agree to the practices described in this Privacy Policy.

2. Information We Collect

Only the data required to provide core chat and account functionality is collected.

2.1 Personal Information

When you sign in or create an account, the following information may be collected:

  • Name (from Google account)
  • Email address
  • User ID (Firebase UID)
  • Profile photo
  • Username (chosen by you)
  • Mobile phone number (entered by you)
  • About / status text

2.2 Messages

The app stores:

  • In-app chat messages
  • Message timestamps
  • Sender and receiver IDs

Messages are used only to provide chat functionality between users.

2.3 Device & Technical Data

The following technical data may be collected:

  • Device identifiers (Firebase-generated IDs)
  • App connectivity status
  • Authentication tokens

This data is used for:

  • Secure login
  • Session management
  • App stability

3. How Information Is Used

The collected data is used strictly for:

  • Account creation and authentication
  • Enabling chat and contact features
  • Displaying user profiles
  • Sending and receiving messages
  • Maintaining app security
  • Preventing misuse or fraud

The data is NOT used for:

  • Advertising
  • Marketing
  • Analytics tracking
  • Selling data

4. Data Sharing

User data is not shared with third parties.

Data is stored securely using:

  • Firebase Authentication
  • Cloud Firestore
  • Firebase Storage

5. Data Storage & Security

  • All data is transmitted over encrypted (HTTPS) connections
  • Firebase security rules protect access
  • Only authenticated users can access their data
  • Passwords are never stored directly

6. Account Deletion & Data Removal

An account can be deleted directly from the app:

  • Open Profile
  • Select Delete Account
  • Confirm deletion

Deletion is immediate and irreversible.

7. Permissions Used

  • Internet – for messaging and authentication
  • Phone call access – only when the call button is tapped
  • Network state – to detect connectivity
  • Storage (Firebase) – for profile images

8. Children’s Privacy

WeTalk is not intended for children under the age of 13. No personal data from children is knowingly collected.

9. Changes to This Policy

This Privacy Policy may be updated when necessary. Any changes will be reflected on this page with an updated effective date.
